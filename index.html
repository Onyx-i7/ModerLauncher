Preguntas Frecuentes

¿Es realmente gratis?
Sí, ModerLauncher es 100% gratuito y siempre lo será. Es un proyecto de código abierto sin ningún tipo de monetización.
¿Puedo confiar en ModerLauncher?
Totalmente. El código está disponible en GitHub para que cualquiera lo revise. No recopilamos datos ni incluimos software malicioso.
¿Dónde lo descargo?
Únicamente desde la página oficial de GitHub Releases. Desconfía de otras fuentes.
¿Funciona con mis mods?
Sí, ModerLauncher es compatible con Forge, Fabric y todos los mods que uses normalmente. Solo instala la versión correspondiente.
¿Necesito instalar Java aparte?
No hace falta. ModerLauncher gestiona Java automáticamente según la versión de Minecraft que elijas.
¿Puedo usar varias cuentas?
Claro, puedes agregar todas las cuentas que quieras y cambiar entre ellas con un simple clic.
