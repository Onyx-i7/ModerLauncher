## 🎮 ¿Qué es ModerLauncher?

**ModerLauncher** es un launcher alternativo y gratuito para Minecraft Java Edition. ModerLauncher ofrece una interfaz moderna, soporte completo para mods con Forge y Fabric, y herramientas de optimización avanzadas para mejorar tu experiencia de juego.

### ¿Por qué elegir ModerLauncher?

ModerLauncher se destaca por:
- **Interfaz elegante**: ModerLauncher tiene un diseño moderno tipo Material Design
- **Optimización**: ModerLauncher permite configurar RAM, CPU y argumentos JVM
- **Compatibilidad**: ModerLauncher soporta Forge, Fabric y todas las versiones de Minecraft
- **Múltiples cuentas**: ModerLauncher gestiona cuentas Premium, Ely.by y Offline
- **Fácil de usar**: ModerLauncher instala versiones y mods con un clic

## 🎮 Cómo Usar ModerLauncher

### Guía de uso de ModerLauncher:

1. **Abre ModerLauncher** por primera vez
2. **Configura tu cuenta** en ModerLauncher
3. Ve a **"Versiones"** en ModerLauncher
4. **Selecciona una versión** en ModerLauncher
5. **Instala** con ModerLauncher (un clic)
6. Configura **RAM y CPU** en ModerLauncher
7. Haz clic en **"PLAY"** en ModerLauncher

### Instalar mods con ModerLauncher:
1. Instala Forge o Fabric con ModerLauncher
2. Ejecuta Minecraft desde ModerLauncher una vez
3. Coloca archivos `.jar` en la carpeta `mods`
4. Reinicia ModerLauncher y selecciona la versión
