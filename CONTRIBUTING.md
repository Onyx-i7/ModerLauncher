# 🛠️ Contribuir a ModerLauncher

¡Gracias por tu interés en contribuir a **ModerLauncher**! Tus ideas, sugerencias y código son bienvenidos. A continuación te dejamos una guía para ayudarte a comenzar.

---

## 📋 Requisitos previos

Antes de contribuir, asegúrate de cumplir con los siguientes requisitos:

- Tener conocimientos básicos de Git y GitHub
- Tener instalado Node.js (si aplica al proyecto)
- Tener instalado un entorno de desarrollo (Visual Studio Code recomendado)
- Tener una cuenta en GitHub

---

## 🧪 Cómo contribuir

1. **Fork** del repositorio
2. Crea una rama (`git checkout -b mi-contribucion`)
3. Realiza tus cambios y haz commit (`git commit -m "Agrega mi funcionalidad"`)
4. Sube la rama (`git push origin mi-contribucion`)
5. Abre un **Pull Request** describiendo claramente qué has hecho

---

## ✅ Buenas prácticas

- Sigue la estructura y estilo del código existente
- Añade comentarios donde sea necesario
- Asegúrate de que tus cambios no rompen funcionalidades existentes
- Incluye capturas o GIFs si tu cambio afecta la interfaz
- Si estás corrigiendo un bug, referencia el issue relacionado (ej: `Closes #5`)
